github上传大文件大于100MB

Posted on 2019-05-21 | Edited on 2019-06-24

使用github作为仓库的时候面向开发者还好，几乎代码都是属于细小零碎文件，但是如果面向大部分文件都是 .mp3、.psd 这类的就要谨慎了，因为这类文件很轻松就超过100MB，然而在github上传协议中，超过100MB的文件需要使用扩展来处理，而不是可以直接上传的。
